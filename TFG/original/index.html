Práctica 1: Introducción a las redes Neuronales Artificiales¶

Gestión del Paciente

En este modulo se implementa la clase paciente, la cual se encargará de gestionar el procesado y calculo de estadísticas de un paciente.

Se implementan distintas funciones para calcular los índices de apnea e hipopnea del paciente, así como para calcular estadísticas sobre el retraso que tienen las señales, y la correlación existente entre las mismas.

Modulo Paciente. Implementación¶

class paciente.Paciente(filename: str, variables_significativas: list)¶

Clase encargada de la gestión de los pacientes.

df (

obj: pd.DataFrame): Dataframe con toda la informacion de las señales del paciente.

filename (

obj: str): Nombre del fichero de donde se ha leido el paciente.

variables_significativas (

obj: list): Lista con la variable significativas del paciente.

correlation_matrix (

obj: np.matrix): Matriz de correlación de las variables significativas del paciente.

sincronizacion_señales (

obj: dict): Diccionario con clave el retraso de las señales que se han estudiado, y valor DataFrame con los resultados del estudio.

time_step (

obj: float): Tiempo entre una muestra y otra. Calculado leyendo el nombre de las columnas del fichero csv.

apnea_index (

obj: float): Índice de apneas por hora del paciente.

hypopnea_index (

obj: float): Índice de hipopneas por hora del paciente.

iah (

obj: float): Índice de apneas-hipopneas por hora del paciente.

almacenar_paciente() → None¶

Almacena la información del paciente serializada.

Almacena el objeto paciente serializado, haciendo uso de la librería Pickle.

calcular_indices_paciente() → None¶

Calcula los índices de apnea, hipopnea y apnea-hipopnea.

Calcula los índices de apnea, hipopnea y apnea-hipopnea, y almacena dicha información en los atributos del paciente.

clean_memory() → None¶

Realiza limpieza de memoria.

Elimina la información relativa al fichero de datos que se ha leido al inicializar el paciente. Función útil cuando se quieren leer múltiples pacientes en RAM pero no se necesita la información leída del fichero.

comprobar_validez_dataset(threshold=0.002) → bool¶

Comprueba que los datos del paciente sean válidado.

Se comprueba que todas las columnas indicadas como variables significativas al inicializar contengan valores. Se tolera un threshold (porcentaje) de valores nan en la columna estudiada, que seran reemplazados por la última muestra válida del mismo.

Parámetros

threshold (float, optional) – Porcentaje de muestras vacías que se tolera que esté con valores nan dentro de una columna. Defaults a 0.002.

Devuelve

True si el paciente se considera válido, al cumplir los requisitos del mínimo número de valores que no pueden ser nan. False en caso contrario.

Tipo del valor devuelto

bool

compute_correlation_matrix(plot_results=False, method='pearson')¶

Calcula la matriz de correlación de las variables significativas.

Se calcula y se almacena la información de la matriz de correlación. Se puede especificar el método de cálculo de la matriz de correlación, y si se desea ver el gráfico del calculo

Parámetros

  • plot_results (bool, optional) – Indica si se quiere mostrar el resultado gráficamente. Defaults a False.

  • method (str, optional) – Indica el cálculo de correlación que se aplica. Los valores posibles son pearson, kendall o spearman. Defaults a pearson.

Devuelve

DataFrame con la matriz de correlación si el parámetro plot_results es False. None en caso contrario.

Tipo del valor devuelto

pd.DataFrame
